# Changelog

All notable changes to this binary release package are documented here.

## [0.1.0] - 2026-06-21

### Added

- Initial closed-source Windows x64 CPU binary package.
- Added `bin/df2d.exe` for running ADR solver cases.
- Added a small smoke-test input at `input/input_parameter_0001.toml`.
- Added `input/input_parameter_reference.toml` as a copyable TOML parameter template.
- Added `run_example.bat` for one-click local smoke testing.
- Added academic evaluation license and distribution notice.
- Added packaged download archive at `release/ADR-windows-x64.zip`.

### Notes

- This release contains no source code.
- The included smoke-test case is only for confirming that the binary runs; it is not intended as a validation or publication case.
- SHA256 checksums are published in the GitHub Release notes.
